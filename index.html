Step-by-step calculation details
Google Sheets formulas to embed
Use these formulas directly inside Google Sheets from your input cells. Format rate outputs as percentages where helpful. The formulas use descriptive variable names such as controlrate and experimentrate because names like p1 and p2 can be interpreted as cell references in Google Sheets.
Recommended Sheets layout: B2 = control conversions C2 = control Audience Reach D2 = experiment conversions E2 = experiment Audience Reach F2 = confidence level, e.g. 0.90 G2 = tail, one of: greater, less, two H2 = target power, e.g. 0.80 I2 = control reach growth per selected period J2 = experiment reach growth per selected period; leave blank to reuse control growth K2 = p-value output L2 = required n / arm output M2 = observed power output N2 = CI lower-bound output O2 = CI upper-bound output P2 = required control N from planning helper Q2 = required experiment N from planning helper R2 = planned test length in days S2 = control unique reach per $1 T2 = experiment unique reach per $1; leave blank to reuse control reach per $1
Control conversion rate: =LET(controlconv,B2,controlreach,C2,controlconv/controlreach) Experiment conversion rate: =LET(experimentconv,D2,experimentreach,E2,experimentconv/experimentreach) Absolute difference, experiment - control: =LET(controlconv,B2,controlreach,C2,experimentconv,D2,experimentreach,E2,controlrate,controlconv/controlreach,experimentrate,experimentconv/experimentreach,experimentrate-controlrate) Relative lift: =LET(controlconv,B2,controlreach,C2,experimentconv,D2,experimentreach,E2,controlrate,controlconv/controlreach,experimentrate,experimentconv/experimentreach,IF(controlrate=0,"",(experimentrate/controlrate)-1))
P-value: =LET(controlconv,B2,controlreach,C2,experimentconv,D2,experimentreach,E2,tailchoice,LOWER(G2),controlrate,controlconv/controlreach,experimentrate,experimentconv/experimentreach,pooledrate,(controlconv+experimentconv)/(controlreach+experimentreach),standarderror,SQRT(pooledrate*(1-pooledrate)*(1/controlreach+1/experimentreach)),zscore,(experimentrate-controlrate)/standarderror,SWITCH(tailchoice,"greater",1-NORM.S.DIST(zscore,TRUE),"less",NORM.S.DIST(zscore,TRUE),"two",2*(1-NORM.S.DIST(ABS(zscore),TRUE)))) Calculated confidence, format the cell as a percent: =1-K2 False-positive risk: =K2 Z-score: =LET(controlconv,B2,controlreach,C2,experimentconv,D2,experimentreach,E2,controlrate,controlconv/controlreach,experimentrate,experimentconv/experimentreach,pooledrate,(controlconv+experimentconv)/(controlreach+experimentreach),standarderror,SQRT(pooledrate*(1-pooledrate)*(1/controlreach+1/experimentreach)),(experimentrate-controlrate)/standarderror)
Difference CI lower bound: =LET(controlconv,B2,controlreach,C2,experimentconv,D2,experimentreach,E2,confidence,F2,controlrate,controlconv/controlreach,experimentrate,experimentconv/experimentreach,difference,experimentrate-controlrate,standarderror,SQRT(controlrate*(1-controlrate)/controlreach+experimentrate*(1-experimentrate)/experimentreach),zcrit,NORM.S.INV(1-(1-confidence)/2),difference-zcrit*standarderror) Difference CI upper bound: =LET(controlconv,B2,controlreach,C2,experimentconv,D2,experimentreach,E2,confidence,F2,controlrate,controlconv/controlreach,experimentrate,experimentconv/experimentreach,difference,experimentrate-controlrate,standarderror,SQRT(controlrate*(1-controlrate)/controlreach+experimentrate*(1-experimentrate)/experimentreach),zcrit,NORM.S.INV(1-(1-confidence)/2),difference+zcrit*standarderror) CI crosses zero? TRUE/FALSE: =AND(N2<=0,O2>=0)
Observed power for the current effect: =LET(controlconv,B2,controlreach,C2,experimentconv,D2,experimentreach,E2,confidence,F2,tailchoice,LOWER(G2),controlrate,controlconv/controlreach,experimentrate,experimentconv/experimentreach,effectdiff,ABS(experimentrate-controlrate),unpooledse,SQRT(controlrate*(1-controlrate)/controlreach+experimentrate*(1-experimentrate)/experimentreach),effectz,effectdiff/unpooledse,zcrit,NORM.S.INV(1-IF(tailchoice="two",(1-confidence)/2,(1-confidence))),(1-NORM.S.DIST(zcrit-effectz,TRUE))+NORM.S.DIST(-zcrit-effectz,TRUE)) False-negative risk for the observed effect: =1-M2 Required n / arm at target power: =LET(controlconv,B2,controlreach,C2,experimentconv,D2,experimentreach,E2,confidence,F2,tailchoice,LOWER(G2),targetpower,H2,controlrate,controlconv/controlreach,experimentrate,experimentconv/experimentreach,effectdiff,ABS(experimentrate-controlrate),averagerate,(controlrate+experimentrate)/2,zcrit,NORM.S.INV(1-IF(tailchoice="two",(1-confidence)/2,(1-confidence))),zpower,NORM.S.INV(targetpower),IF(effectdiff=0,"",ROUNDUP(((zcrit*SQRT(2*averagerate*(1-averagerate))+zpower*SQRT(controlrate*(1-controlrate)+experimentrate*(1-experimentrate)))^2)/(effectdiff^2),0))) Additional Audience Reach needed in the smaller arm: =IF(L2="","",MAX(0,L2-MIN(C2,E2)))
Reach-growth periods needed: =IFERROR(MAX(0,ROUNDUP((L2-C2)/I2,0),ROUNDUP((L2-E2)/IF(J2="",I2,J2),0)),"") Reach-growth days needed when your growth input is daily: =IFERROR(MAX(0,ROUNDUP((L2-C2)/I2,0),ROUNDUP((L2-E2)/IF(J2="",I2,J2),0)),"") Reach-growth days needed when your growth input is weekly: =IFERROR(7*MAX(0,ROUNDUP((L2-C2)/I2,0),ROUNDUP((L2-E2)/IF(J2="",I2,J2),0)),"") Reach-growth days needed when your growth input is monthly: =IFERROR(30.4375*MAX(0,ROUNDUP((L2-C2)/I2,0),ROUNDUP((L2-E2)/IF(J2="",I2,J2),0)),"")
Planning-helper spend outputs: Control daily spend: =IFERROR(P2/S2/R2,"") Experiment daily spend: =IFERROR(Q2/IF(T2="",S2,T2)/R2,"") Total daily spend: =SUM(IFERROR(P2/S2/R2,0),IFERROR(Q2/IF(T2="",S2,T2)/R2,0)) Control total spend: =IFERROR(P2/S2,"") Experiment total spend: =IFERROR(Q2/IF(T2="",S2,T2),"") Total planned spend: =SUM(IFERROR(P2/S2,0),IFERROR(Q2/IF(T2="",S2,T2),0))
Method: pooled z-test for the null hypothesis that both groups have equal conversion rates. The confidence interval for the observed difference uses the unpooled standard error. For media PSA, ghost, UA, or RE tests, use unique Audience Reach as the denominator, not impression counts.
